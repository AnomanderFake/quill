That's it! The pipeline will:
1. Refine the prose using Le Guin principles
2. Review it against quality standards
3. Return refined text + detailed feedback

Expected outcome for the example above:
- Much shorter (probably 50% reduction)
- Physical details instead of emotion labels
- No thought-reporting ("I wondered if...")
- No self-analysis ("she represented constancy")
- Trust the images to carry the meaning
